Logical Model: Patient Enrollment Program

Official URL: http://example.org/StructureDefinition/PatientEnrollmentProgram Version: 0.1.0
Draft as of 2024-12-03 Computable Name: PatientEnrollmentProgram

A tracker program to manage patient information, designed for mapping to the FHIR Patient resource. This program captures key demographic and identifying details required for patient enrollment.

Usage:

  • This Logical Model Profile is not used by any profiles in this Implementation Guide
